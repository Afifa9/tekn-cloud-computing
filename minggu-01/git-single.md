## Membuat Repo Di Account Sendiri

## Membuat Repo
Untuk membuat repo, gunakan langkah-langkah berikut:

1.  Klik tanda **+** pada bagian atas setelah login, pilih **New repository**

![Create New Repository menu](![gmb22](https://user-images.githubusercontent.com/114986359/224526255-c038a599-00ac-46b5-ab36-081ac829778f.png)
)

2.  Isikan nama, keterangan, serta lisensi. Jika dikehendaki, bisa membuat repo **Private**

![Isian repo baru](![gmb23](https://user-images.githubusercontent.com/114986359/224526264-5b5f19b4-fd39-4334-a503-43446c9c99c6.png)
)
