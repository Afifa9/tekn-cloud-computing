## Membuat Repo
Untuk membuat repo, gunakan langkah-langkah berikut:

1.  Klik tanda **+** pada bagian atas setelah login, pilih **New repository**


2.  Isikan nama, keterangan, serta lisensi. Jika dikehendaki, bisa membuat repo **Private**
